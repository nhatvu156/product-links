# Tổng kết dự án: Product Link Hub V2

Dự án này là một ứng dụng Web lưu trữ liên kết sản phẩm chuyên nghiệp, được thiết kế theo phong cách tối giản của Apple.

## 🚀 Công nghệ sử dụng
- **React 18** + **Vite**: Nền tảng xây dựng ứng dụng nhanh và hiện đại.
- **Tailwind CSS v4**: Xử lý giao diện và Dark mode bằng biến CSS thế hệ mới.
- **Framer Motion**: Tạo các hiệu ứng chuyển cảnh mượt mà và cử chỉ vuốt (swipe) trên di động.
- **Lucide React**: Bộ icon tối giản, đồng bộ.
- **Local Storage**: Lưu trữ dữ liệu trực tiếp trên trình duyệt của người dùng.

## ✨ Tính năng chính
1. **Giao diện Apple Minimal**: Hỗ trợ đầy đủ Chế độ Sáng/Tối (Light/Dark Mode).
2. **Bộ lọc thông minh**:
   - Lọc theo Danh mục (Categories) dạng thanh cuộn ngang.
   - Tìm kiếm theo Tên, Mô tả hoặc Thẻ (Tags).
   - Sắp xếp theo Mới nhất hoặc A-Z.
3. **Trải nghiệm di động**: Hỗ trợ cử chỉ vuốt sang trái để Xóa sản phẩm.
4. **Quản lý linh hoạt**: Thêm, Xóa và Sao chép liên kết chỉ với một cú chạm.

## 🌐 Triển khai (Deployment)
- **Nền tảng**: GitHub Pages.
- **Cơ chế**: Tự động hóa qua GitHub Actions.
- **Địa chỉ Web**: [https://nhatvu156.github.io/product-links/](https://nhatvu156.github.io/product-links/)

## 🛠️ Hướng dẫn bảo trì
- Để chạy máy ảo phát triển: `npm run dev`
- Để cập nhật tính năng: Chỉ cần `git add .`, `git commit` và `git push origin main`. GitHub sẽ tự động cập nhật website cho bạn sau khoảng 1-2 phút.
